## Start

Create `.infura` and `.secret` files. Install the dependencies:

```bash
$ yarn
```

## Tests

```bash
$ yarn test
```

## Coverage

```bash
$ yarn coverage
```

## Deploying

Deploy to Kovan:

```bash
$ yarn oz deploy
```
